Ressources — Étape 0 : Installation des outils

Résumé structuré — Étape 0

Objectif : Préparer votre environnement pour coder en Python (PC ou Android) et démarrer des projets Flask/Django.

  • Installer : les outils nécessaires selon votre appareil :
    • Ordinateur (Windows/macOS/Linux) : Python, VS Code ou PyCharm, Git pour versionner vos projets.
    • Smartphone Android : Termux (émulateur terminal), Python et Git via Termux, éditeur de code léger (nano, vim ou micro).
  • Pré-requis :
    • Ordinateur ou smartphone Android
    • Connexion Internet pour télécharger les outils
    • Un peu de patience pour configurer l’environnement la première fois
  • Résultat attendu :
    • Pouvoir lancer python depuis votre terminal
    • Créer et activer un environnement virtuel (venv)
    • Versionner vos projets avec Git et éventuellement pousser sur GitHub/GitLab
  • Durée estimée : ~20–30 minutes selon votre expérience et votre appareil.

Chaque bloc de commande contient un commentaire (lignes commençant par #) expliquant l’action, et le bouton Copier à droite pour coller directement la commande dans votre terminal.

Outils pour ordinateur

  • Python : interpréteur pour exécuter vos scripts.
  • VS Code / PyCharm : éditeurs pour écrire, tester et déboguer votre code.
  • Git : gestionnaire de version pour sauvegarder vos projets et collaborer.

Outils pour smartphone Android

  • Termux : terminal qui permet d’installer Python et Git sur Android.
  • Python : pour exécuter vos scripts directement dans Termux.
  • Éditeur de code léger : nano, vim ou micro pour écrire vos scripts dans Termux.
  • Git : pour versionner vos projets et les synchroniser avec un dépôt distant.

Outils principaux

  • Python — langage utilisé pour tous les exercices. Téléchargement officiel.
  • Éditeur — VS Code (recommandé) ou PyCharm. VS Code — PyCharm.
  • Git — gestion de versions : git-scm.
  • Termux (Android) — terminal Linux pour Android : F-Droid / Termux.

Commandes (copiables)

Termux (Android)

# Mettre à jour les paquets

pkg update && pkg upgrade
# Installer Python

pkg install python
# Installer Git (optionnel)

pkg install git
# Vérifier Python

python --version

PC / macOS / Linux

# Vérifier Python (PC/Mac/Linux)

python --version  # ou python3 --version
# Dans VS Code : ouvrir la palette de commandes

Ctrl+Shift+P → Python: Select Interpreter
# Windows (winget) — installer VS Code

winget install --id Microsoft.VisualStudioCode
# Vérifier Git

git --version
# Initialiser un dépôt local

git init
# Configurer nom et e-mail (remplacez par vos infos)

git config --global user.name "Votre Nom" git config --global user.email "email@example.com"

Environnements virtuels & exécution

# Linux / macOS

python -m venv env source env/bin/activate
# Windows (PowerShell)

python -m venv env

Dans PowerShell, exécutez :

.\env\Scripts\Activate.ps1
# Exécuter un script Python

python main.py
